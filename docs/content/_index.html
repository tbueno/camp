Your all-in-one development environment manager

{{< blocks/link-down color="info" >}}
{{< /blocks/cover >}} {{% blocks/lead color="secondary" %}} Camp is a command-line tool designed to help developers manage their isolated development environments. Built with Go and powered by Nix, Camp provides essential system information and environment management utilities. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-lightbulb" title="Simple Configuration" %}} Manage your development environment with a single YAML file. Configure environment variables, packages, and external Nix flakes all in one place. {{% /blocks/feature %}} {{% blocks/feature icon="fa-boxes" title="Package Management" %}} Declaratively manage Nix packages across macOS and Linux. Install and update packages with a simple rebuild command. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-nix" title="Nix-Powered" %}} Built on Nix for reproducible, reliable environments. Works with nix-darwin on macOS and home-manager on Linux. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section >}}

Get started with Camp in minutes
