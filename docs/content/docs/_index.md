Welcome to the Camp documentation!

Camp is a command-line tool that helps developers manage isolated development environments using Nix. Whether you're setting up a new machine or managing multiple projects, Camp simplifies environment configuration and package management.

## What is Camp?

Camp provides:

- **System Information**: Get detailed information about your system architecture and operating system
- **Environment Configuration**: Manage custom environment variables via `camp.yml`
- **Package Management**: Declaratively install and manage Nix packages
- **Nix Integration**: Rebuild development environments using Nix configurations
- **Flake Support**: Integrate external Nix flakes for extensibility
- **Platform Support**: Works on both macOS (via nix-darwin) and Linux (via home-manager)
